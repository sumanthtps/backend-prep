```java
import org.junit.jupiter.api.BeforeEach;
import org.junit.jupiter.api.Test;

class MyMathTest {

    private MyMath math;

    @BeforeEach
    void setup() {
        math = new MyMath();  // Runs before each test
    }

    @Test
    void testCalculateSum() {
        assertEquals(6, math.calculateSum(new int[]{1, 2, 3}));
    }

    @Test
    void testCalculateSum_EmptyArray() {
        assertEquals(0, math.calculateSum(new int[]{}));  // Edge case: empty array
    }
}
```

âœ… **Improvements:**

- **`@BeforeEach`** â†’ Initializes the `MyMath` object **before each test**.
- **Added an edge case** â†’ Tests what happens with an **empty array** (`{}`).
